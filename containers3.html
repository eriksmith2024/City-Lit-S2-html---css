Structural Tags in HTML

To allow us to create areas in our page to contain content that are semantic, we got some new tags in HTML 5

The Header Element

We can use this to define the header element of our page

it might include content such as the site title, possibly the navigation bar, tagline, logos, branding, search account icon / avatar, shopping cart

The Nav Element

We can use this to contain any significant menu of links. The nav element can appear on ts own or nested within other elements

The Main Element

Use this to define the main content area. It should only include content that is unique to this page.

The Section Element

Use this to define an area of content or the start of a new topic - used for discrete areas

The Article Element

Use this to define an area of content that would still make sense if we took it out of the page and put it somewhere else.

The Footer Element

This should contain information at the end of the page. It may contain copyright information, contact details, physical address, social networks links, additional nav i.e. map, t&cs, mail sign ups, secondary navigation
