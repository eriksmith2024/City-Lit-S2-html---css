Box Model Revision

The box model helps us to understand how the broswer renders the HTML elements

All elements can be considered to be boxes. They all have default CSS values.

Width

As soon as we specify the width of the element the default is overridden

Margin

Margins are the space around the outside of the box

Padding

Padding is space applied inside the box

Border

Borders are applied between the margin and padding

Two Values

Two values for padding, margin or border will apply to the top and bottom and the second to the left or right
