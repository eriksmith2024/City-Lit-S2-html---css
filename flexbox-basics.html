Flexible Box Model

When the browser starts to render a web page it will read through the html elements and create them in the same order as they appear in the code.

This is called the flow and the browser will stack each element vertically in the same order as the code

Flexbox Layout

One way of rearranging our boxes is to put them in to flex box mode which allows us to flow our boxes in different ways.
